# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Add Patient modal with comprehensive form and validation
- Icon improvements for patient status indicators (Active → HiOutlineUserGroup, Referred → HiOutlineArrowUpRight)
- Prepared project for GitHub deployment with CI/CD workflows

### Changed
- Improved Add Patient form styling with color-coded sections
- Enhanced modal design with better typography and spacing

### Fixed
- Fixed SQL syntax error in patient form height field

## [1.0.0] - 2026-03-22

### Added
- Initial release
- ✨ **Dashboard Overview** - Real-time statistics and activity charts
- 📁 **Patient Management** - Full CRUD operations with detailed medical records
- 🔬 **MRI Upload & Analysis** - Scan upload and AI-powered tumor detection REST integration
- 🤖 **Model Training** - Interactive training interface with real-time metrics
- 📊 **Evaluation Metrics** - Comprehensive model evaluation dashboards
- 📋 **Reports** - Report generation and management
- 👥 **User Management** - Role-based access control (Super Admin, Radiologist, Researcher)
- 🔐 **Authentication** - Token-based auth with refresh token flow
- 🎨 **Modern UI** - Tailwind CSS with responsive design
- 📱 **Mobile Responsive** - Works on all device sizes
- 🌙 **Dark Mode Ready** - Prepared for theme support

### Technical Stack
- React 18.3
- TypeScript 5.6
- Vite 6.0
- Tailwind CSS 3.4
